🔑 Paso 1 — Conectá tu cuenta de Binance
Para que el bot opere con tu plata, necesitamos conectarnos a tu cuenta de Binance a través de una API Key. Esto es como darle una "llave" al bot para que pueda comprar y vender por vos, pero sin poder retirar tu dinero.
¿No tenés cuenta en Binance? Creá una gratis en binance.com y depositá USDT (mínimo $50 recomendado).
1
Entrá a la gestión de API
Abrí Binance → Gestión de API o andá a tu perfil → "Gestión de API".
2
Creá una nueva API Key
Hacé clic en "Crear API", elegí "Generado por el sistema" y ponele un nombre como "AutoTrader Bot".
3
Configurá los permisos
Activá solo "Habilitar Spot y Trading con Margen". ¡NUNCA actives la opción de retiros! Así tu plata queda siempre segura en tu cuenta.
4
Copiá las claves y pegalas abajo
Binance te va a mostrar dos textos largos: la API Key y el Secret Key. Copialos y pegalos en los campos de abajo. El Secret solo se muestra una vez — guardalo en un lugar seguro.
Es un texto largo con letras y números. Ejemplo: 7xKq2B...
Binance lo muestra solo una vez al crear la API. Si lo perdiste, tenés que crear una API nueva.
🔒 Tu seguridad es nuestra prioridad. Las API Keys solo permiten operar — nunca retirar fondos. Tu plata queda siempre en tu cuenta de Binance bajo tu control total.
💰 Paso 2 — ¿Cuánto querés operar?
Elegí el monto en USDT que el bot va a usar para operar. Es el capital que tenés disponible en tu cuenta de Binance. Podés empezar con poco y aumentar después.
Mínimo recomendado: $50 USDT. Asegurate de tener este monto en tu cuenta de Binance.
⚙️ Plan Basic: El bot opera automáticamente con parámetros optimizados — Stop Loss 1.5%, Take Profit 2.5%, par BTC/USDT. No necesitás configurar nada más.

❓ Preguntas frecuentes
¿Es seguro dar mi API Key? ▸
Sí. La API Key solo permite operar (comprar y vender). No permite retirar fondos. Tu dinero queda siempre en tu cuenta de Binance bajo tu control total. Además, podés desactivar la API en cualquier momento desde Binance.
¿Cuánto dinero necesito para empezar? ▸
Recomendamos un mínimo de $50 USDT para que el bot pueda operar cómodamente. Con menos, las comisiones de Binance pueden afectar la rentabilidad.
¿Puedo pausar el bot cuando quiera? ▸
Sí, podés detener y reactivar el bot en cualquier momento desde esta página. Cuando lo pausás, no se realizan más operaciones hasta que lo reactives.
¿El bot puede perder mi dinero? ▸
El trading de criptomonedas siempre implica riesgo. El bot usa Stop Loss para limitar las pérdidas por operación, pero no puede garantizar ganancias. Invertí solo lo que estés dispuesto a perder.
¿Cómo veo mis resultados? ▸
En el Dashboard podés ver todas las operaciones del bot en tiempo real: ganancias, pérdidas, win rate, gráficos y más. También recibís alertas por Telegram si lo configurás.
⚠️ Aviso de riesgo: El trading de criptomonedas es volátil y especulativo. Los resultados pasados no garantizan rendimientos futuros. Podés perder parte o la totalidad de tu capital. AutoTrader Pro no es asesoría financiera.
